# NRC Marathon Tracker

Track your progress through Nike Run Club's **18-week Audio Guided Marathon Training Plan**.

## Features

- Full 18-week schedule (recovery, speed, and long runs)
- Pace chart with personalized targets
- Log runs (distance, time, pace, feeling, notes)
- Mark workouts complete and see weekly progress
- Race day countdown

## Quick start

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Official plan

This site is a standalone marathon training tracker. The official PDF is not included in this repository.

## Data

All data is stored in your browser (`localStorage`). Nothing is sent to a server.
